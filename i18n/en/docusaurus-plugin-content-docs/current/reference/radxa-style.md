# Radxa Style Reference

This project references the documentation structure of Radxa Docs, without copying its code or brand visuals.

## Reference points

| Design point | Implementation in this project |
| --- | --- |
| Top navigation | Product groups, all-docs entry, search, and language switcher |
| Left sidebar | Autogenerated from the documentation folder structure |
| Content layout | Clear headings, tables, code blocks, images, and admonitions |
| Right table of contents | Generated from page headings |
| Deployment | GitHub Actions builds and deploys the static site |

## Document URL generation

```text
docs/core-board/rk3588-core-board/product-specification.md
```

generates:

```text
/docs/core-board/rk3588-core-board/product-specification
```

The English page uses the same relative path under `i18n/en/docusaurus-plugin-content-docs/current/` and is served under `/en/docs/...`.
