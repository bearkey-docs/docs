# Project Overview

This repository builds a Docusaurus documentation website from Markdown / MDX files stored in GitHub.

## Workflow

```text
GitHub Markdown / MDX
        │
        ▼
Docusaurus build
        │
        ▼
Static HTML / CSS / JavaScript
        │
        ▼
GitHub Pages deployment
```

## Repository structure

| Path | Purpose |
| --- | --- |
| `docs/` | Chinese documentation source |
| `i18n/en/docusaurus-plugin-content-docs/current/` | English documentation source |
| `src/pages/` | Custom pages |
| `src/css/custom.css` | Site styles |
| `sidebars.js` | Autogenerated docs sidebar |
| `docusaurus.config.js` | Site and navbar configuration |
| `.github/workflows/deploy.yml` | Build and deploy workflow |

:::tip
Update documentation by editing Markdown / MDX files in the language-specific folder.
:::
